# SKILL: Video → LinkedIn Posts (Marc Bau)

## Quick Reference

### POSICIONAMIENTO

```
"Implemento IA que entiende tu negocio"
"Recupera 20 horas a la semana en 30 días"

Target: Empresas de servicios B2B (1-10M€)
Ticket: €3.000 - €8.000
```

---

### 1. EXTRAER TEMAS (2-4 por video)

```
Para cada tema preguntar:
☐ ¿Relevante para CEO/director de empresa de servicios?
☐ ¿Se explica sin jerga técnica?
☐ ¿Conecta con "perder tiempo en tareas repetitivas"?
☐ ¿Se puede relacionar con "recuperar horas"?
```

### 2. MULTIPLICAR POR ESTRATEGIAS

| Estrategia | Visual | CTA |
|------------|--------|-----|
| **Historia Personal** | Foto tuya trabajando | Suave o ninguno |
| **Educativo/Framework** | Diagrama simple | "Guarda esto" |
| **Contrarian** | Sin imagen | Pregunta provocadora |
| **Caso/Resultado** | Números destacados | "¿Te suena?" |
| **Lead Magnet** | Mockup del recurso | "Comenta [X]" |

### 3. ICP RECORDATORIO

```
QUIÉN: CEO/Fundador de empresa de servicios B2B (1-10M€)
PIENSA: "Pierdo mucho tiempo en tareas repetitivas"
BUSCA: Recuperar tiempo sin contratar más gente
NO QUIERE: Jerga técnica, promesas exageradas, soluciones genéricas
```

### 4. PALABRAS

```
❌ PROHIBIDAS:
Agentes, MCP, skills, workflows, RAG, embeddings
Reimaginar, revolucionar, transformar, ecosistema
"Reemplaza tu equipo", "$200K team", "10x"
"En la era de la IA", "el futuro del trabajo"

✅ USAR:
Tiempo, horas, proceso, repetitivo
Funciona, ahorra, reduce, recupera
Tu negocio, tu forma de trabajar, personalizado
"He visto", "esta semana", "un cliente"
```

### 5. HOOKS QUE FUNCIONAN

```
NÚMERO ESPECÍFICO:
"20 horas a la semana."
"De 4 horas a 30 minutos."

PREGUNTA:
"¿Cuántas horas pierdes en tareas que podrían hacerse solas?"

OBSERVACIÓN:
"La mayoría de CEOs que conozco..."
"Hay un patrón que veo una y otra vez..."

CONTRARIAN:
"No necesitas más herramientas de IA."
"ChatGPT no funciona para tu negocio. Y es normal."

HISTORIA:
"Esta semana un cliente me preguntó algo..."
"Ayer pasé 3 horas configurando algo que..."
```

### 6. LEAD MAGNETS

```
1. "5 Procesos que Toda Empresa Debería Automatizar" → PROCESOS
2. "Checklist: ¿Dónde Pierde Tiempo tu Equipo?" → CHECKLIST
3. "Por qué ChatGPT no Funciona para tu Negocio" → CHATGPT
```

### 7. CTAs

```
PARA AWARENESS:
"Sígueme para más sobre IA práctica para negocios."

PARA LEAD MAGNET:
"Si quieres el checklist, comenta 'CHECKLIST'."

PARA DIAGNÓSTICO:
"Si quieres una conversación de 20 minutos sobre 
dónde tiene sentido automatizar en tu empresa:
Escríbeme 'Diagnóstico'."
```

### 8. OUTPUT POR POST

```
- Texto completo (150-300 palabras)
- Estrategia usada
- Visual recomendado
- CTA específico
- Lead magnet (si aplica)
```
