# SKILL: Video a Posts de LinkedIn para Marc Bau

## Propósito

Transformar transcripciones de videos de YouTube en posts de LinkedIn que generen leads mid-market (3-8k€) para Marc Bau, consultor independiente de implementación de IA personalizada.

**Posicionamiento central:** "Implemento IA que entiende tu negocio | Recupera 20 horas a la semana en 30 días"

---

```

- Asistentes entrenados con información específica del cliente

ESCALERA DE OFERTAS:

- Sistema Equipo IA (€5.000-8.000) - 30 días ← CORE

EXPERIENCIA/REFERENCIAS:

DREAM OUTCOME QUE VENDO:

QUÉ LE PREOCUPA:

NO:

## PARTE 2: PROCESO DE EXTRACCIÓN DE TEMAS

### Paso 1: Analizar la transcripción

Al recibir una transcripción, identificar:

```
1. TEMA PRINCIPAL
   ¿De qué habla el video en esencia?
   (1 frase)

2. SUBTEMAS/INSIGHTS
   ¿Qué puntos específicos desarrolla?
   (Listar 3-6 máximo)

3. EJEMPLOS/CASOS
   ¿Qué historias o ejemplos concretos usa?
   (Listar todos)

4. FRAMEWORKS/MODELOS
   ¿Hay algún framework, lista o modelo que se pueda extraer?
   (Listar si hay)

5. FRASES POTENTES
   ¿Qué frases del video funcionarían como hooks?
   (Listar 3-5)
```

### Paso 2: Filtrar para relevancia

Preguntar para cada subtema:

```
☐ ¿Es relevante para un directivo de empresa tradicional?
☐ ¿Se puede explicar sin jerga técnica?
☐ ¿Tiene aplicación práctica para el ICP?
☐ ¿Conecta con algún dolor/preocupación del ICP?

Si no pasa al menos 3 de 4 → Descartar o reformular
```

### Paso 3: Seleccionar temas finales

```
REGLA: 2-4 temas por video

- Si el video es muy denso → 4 temas máximo
- Si el video es enfocado → 2-3 temas
- Cada tema debe poder sostenerse solo (no depender de los otros)
```

## PARTE 3: MATRIZ DE ESTRATEGIAS DE POSTS

### Las 5 Estrategias

```
┌─────────────────────────────────────────────────────────────────┐
│                    5 ESTRATEGIAS DE POST                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. HISTORIA PERSONAL                                           │
│     "Cómo lo hice yo / Qué vi esta semana"                     │
│     → Foto personal o de trabajo                                │
│     → Tono: cercano, vulnerable, real                          │
│     → CTA: suave o ninguno                                      │
│                                                                  │
│  2. EDUCATIVO/FRAMEWORK                                         │
│     "El modelo / Los X pasos / El framework"                   │
│     → Imagen con diagrama o lista visual                        │
│     → Tono: profesor, claro, estructurado                      │
│     → CTA: "Guarda esto"                                        │
│                                                                  │
│  3. CONTRARIAN/PROVOCADOR                                       │
│     "Lo que todo el mundo hace mal / La verdad incómoda"       │
│     → Sin imagen o imagen de contraste                         │
│     → Tono: directo, sin miedo, retador                        │
│     → CTA: pregunta que invite a comentar                      │
│                                                                  │
│  4. CASO/RESULTADO                                              │
│     "Cliente X consiguió Y / Esto es lo que pasó"              │
│     → Imagen con números o antes/después                        │
│     → Tono: factual, específico, con números                   │
│     → CTA: "¿Te suena familiar?"                               │
│                                                                  │
│  5. LEAD MAGNET                                                 │
│     "He creado X para solucionar Y"                            │
│     → Imagen del recurso o mockup                              │
│     → Tono: generoso, directo al valor                         │
│     → CTA: "Comenta [PALABRA] y te lo envío"                   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Matriz de Combinación

```
TEMAS (del video) × ESTRATEGIAS = POSTS

Ejemplo:
- 3 temas × 5 estrategias = 15 posts posibles
- Pero no todas las combinaciones funcionan
- Seleccionar las mejores: 6-10 posts por video

COMBINACIONES QUE SUELEN FUNCIONAR:

Tema técnico/framework → Educativo, Lead Magnet
Tema de error/fallo → Historia Personal, Contrarian
Tema de resultado → Caso/Resultado, Lead Magnet
Tema filosófico/mindset → Contrarian, Historia Personal
```

## PARTE 4: FORMATOS DE POST POR ESTRATEGIA

### 1. HISTORIA PERSONAL

```
ESTRUCTURA:
───────────
[Hook: Situación concreta]
[Contexto: Qué pasó]
[Desarrollo: El problema/desafío]
[Giro: Lo que aprendí/descubrí]
[Cierre: Reflexión aplicable]

VISUAL: Foto personal (tuya trabajando, en reunión, en oficina)
LONGITUD: 150-250 palabras
CTA: Suave ("¿Te ha pasado algo parecido?") o ninguno

EJEMPLO DE HOOK:
"La semana pasada un cliente me preguntó algo que me dejó pensando."
"Ayer perdí 3 horas en una reunión que podría haber sido un email."
"El otro día metí la pata con un cliente. Así fue."
```

### 2. EDUCATIVO/FRAMEWORK

```
ESTRUCTURA:
───────────
[Hook: Problema común o pregunta]
[Introducción: Por qué importa]
[Framework: Los X pasos/elementos]
[Desarrollo breve de cada punto]
[Cierre: Cómo aplicarlo]

VISUAL: Diagrama, lista visual, o infografía simple
LONGITUD: 200-300 palabras
CTA: "Guarda esto para cuando lo necesites"

EJEMPLO DE HOOK:
"3 preguntas antes de implementar cualquier IA en tu empresa:"
"El framework que uso para evaluar si un proceso merece automatizarse:"
"Esto es lo que separa proyectos de IA que funcionan de los que no:"
```

### 3. CONTRARIAN/PROVOCADOR

```
ESTRUCTURA:
───────────
[Hook: Afirmación contraria al consenso]
[Desarrollo: Por qué el consenso está mal]
[Evidencia: Ejemplos o datos]
[Matiz: Cuándo sí aplica el consenso]
[Cierre: La verdad incómoda]

VISUAL: Ninguna, o imagen de contraste
LONGITUD: 150-250 palabras
CTA: Pregunta provocadora que invite a comentar

EJEMPLO DE HOOK:
"La mayoría de proyectos de IA no deberían existir."
"Tu empresa probablemente no necesita IA ahora mismo."
"El problema no es la tecnología. Nunca lo fue."
```

### 4. CASO/RESULTADO

```
ESTRUCTURA:
───────────
[Hook: Resultado con número]
[Contexto: Situación inicial del cliente]
[Problema: Qué no funcionaba]
[Solución: Qué hicimos (sin jerga)]
[Resultado: Números concretos]
[Aprendizaje: Qué se puede extraer]

VISUAL: Números destacados, gráfico simple, o antes/después
LONGITUD: 200-300 palabras
CTA: "¿Te suena familiar?" o "¿Tienes un proceso parecido?"

EJEMPLO DE HOOK:
"De 4.000€/mes a 400€/mes. Mismo resultado."
"15 horas semanales. Eso ahorramos en un solo proceso."
"ROI del 420% en 3 meses. Así lo conseguimos."
```

### 5. LEAD MAGNET

```
ESTRUCTURA:
───────────
[Hook: Problema específico que resuelve]
[Agitación: Por qué es frustrante]
[Solución: Qué es el recurso]
[Contenido: Qué incluye (3-5 bullets)]
[Prueba: Por qué funciona]
[CTA: Cómo conseguirlo]

VISUAL: Mockup del recurso, o imagen representativa
LONGITUD: 150-200 palabras
CTA: "Comenta [PALABRA] y te lo envío" + "Conecta conmigo primero"

EJEMPLO DE HOOK:
"He creado la checklist que uso antes de cada proyecto de IA."
"Este framework me ha ahorrado meses de trabajo."
"El documento que envío a clientes antes de la primera reunión."
```

## PARTE 5: GUÍA DE LEAD MAGNETS PARA MARC BAU

### Principios Adaptados

```
PRINCIPIO 1: UN PROBLEMA, UNA SOLUCIÓN
─────────────────────────────────────
❌ "Guía completa de IA para empresas" (demasiado amplio)
✅ "Las 5 preguntas que hago antes de empezar cualquier proyecto de IA"

─────────────────────────────────────
❌ "Cómo ahorrar tiempo con IA"
✅ "El proceso que redujo 40 horas semanales a 3 en una agencia"

PRINCIPIO 3: INMEDIATAMENTE APLICABLE
─────────────────────────────────────
El lector debería poder usar el recurso HOY, sin ayuda externa.
Si necesita contratar a alguien para implementarlo, no es un buen lead magnet.

PRINCIPIO 4: DEMUESTRA TU METODOLOGÍA
─────────────────────────────────────
El lead magnet debe dar una "muestra" de cómo trabajas.
Si les gusta el recurso, confiarán en tu trabajo.

## PARTE 6: INDICACIONES VISUALES

### Qué usar en cada tipo de post

```
┌────────────────┬─────────────────────────────────────────────────┐
│ ESTRATEGIA     │ VISUAL RECOMENDADO                              │
├────────────────┼─────────────────────────────────────────────────┤
│ Historia       │ Foto personal: tú trabajando, en reunión,       │
│ Personal       │ en la oficina, con el equipo, en evento.        │
│                │ Selfie natural > foto de stock.                 │
├────────────────┼─────────────────────────────────────────────────┤
│ Educativo/     │ Diagrama simple, lista visual, framework        │
│ Framework      │ dibujado, captura de pantalla anotada.          │
│                │ Canva o incluso foto de pizarra/papel.          │
├────────────────┼─────────────────────────────────────────────────┤
│ Contrarian     │ Sin imagen (el texto es el protagonista)        │
│                │ O imagen de contraste (ej: comparativa)         │
├────────────────┼─────────────────────────────────────────────────┤
│ Caso/          │ Números grandes destacados, gráfico simple      │
│ Resultado      │ de antes/después, captura de resultados.        │
├────────────────┼─────────────────────────────────────────────────┤
│ Lead Magnet    │ Mockup del recurso (PDF, Excel visible),        │
│                │ imagen representativa del contenido.            │
└────────────────┴─────────────────────────────────────────────────┘
```

### Reglas generales

```
1. MENOS ES MÁS
   Una imagen potente > múltiples imágenes mediocres
   A veces ninguna imagen > imagen de stock genérica

2. AUTENTICIDAD > PRODUCCIÓN
   Foto de móvil auténtica > diseño de Canva perfecto
   LinkedIn premia lo real, no lo pulido

3. EL TEXTO ES REY
   La imagen complementa, no sustituye
   Si el post no funciona sin imagen, reescríbelo

4. COHERENCIA VISUAL
   Mantener un estilo reconocible
   No cambiar de estética cada post
```

## PARTE 7: PROCESO COMPLETO (RESUMEN)

### Input → Output

```
INPUT:
- Transcripción de video de YouTube

OUTPUT:
- Lista de temas extraídos (2-4)
- Posts generados (6-10)
- Para cada post:
  • Texto completo
  • Estrategia usada
  • Visual recomendado
  • CTA específico
  • Si aplica lead magnet, cuál
```

### Checklist por Post

```
☐ ¿El hook engancha en las primeras 2 líneas?
☐ ¿Está libre de jerga técnica innecesaria?
☐ ¿Un directivo de empresa tradicional lo entendería?
☐ ¿Conecta con algún dolor/preocupación del ICP?
☐ ¿Tiene un CTA claro (o intencionalmente no lo tiene)?
☐ ¿La longitud es apropiada para la estrategia?
☐ ¿El visual recomendado tiene sentido?
☐ ¿Suena a persona real, no a ChatGPT?
```

### Palabras/Frases Prohibidas

```
EVITAR:
- "Reimaginar", "revolucionar", "transformar digitalmente"
- "Soluciones de IA", "ecosistema", "sinergia"
- "Potenciar", "apalancar", "escalar exponencialmente"
- "En el mundo actual", "en la era de la IA"
- Cualquier cosa que suene a presentación corporativa

PREFERIR:
- "Funciona", "ahorra", "reduce", "consigue"
- "Proceso", "sistema", "equipo", "resultado"
- "He visto", "nos pasó", "aprendimos"
- Lenguaje que usarías tomando un café con el cliente
```
